# Travel Wallet Static

Travel Wallet Static is a responsive web application that helps users manage and track their travel expenses through a simple and user-friendly interface.

## Features

- Responsive design
- Expense tracking
- Budget management
- Easy-to-use interface
- Works on desktop and mobile devices

## Technologies Used

- HTML
- CSS
- JavaScript

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/Saran2004p/Travel_Wallet_Static.git
```

2. Open the project folder.

3. Run `index.html` in your browser.
